**Card** — base surface for the black canvas: dark fill, hairline border, soft shadow.

```jsx
<Card>Contenido</Card>
<Card interactive glow>Resaltado</Card>
```

`interactive` adds a hover lift + pink border; `glow` keeps the pink glow on at rest. Use it as the container for custom content; `ProjectCard` and `CertCard` build on the same look.
