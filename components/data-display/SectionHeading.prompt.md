**SectionHeading** — opens every portfolio section with an eyebrow tick, large Syne display title, and optional subtitle.

```jsx
<SectionHeading
  eyebrow="Proyectos"
  title="Trabajo seleccionado"
  subtitle="Una muestra de lo que disfruto crear."
/>
```

`align` can be `left` (default) or `center`. The eyebrow renders a short pink gradient tick before the label.
